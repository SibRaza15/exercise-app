##Exercise App

This is a full-stack app that utilizes the MEARN stack. This app allows the user to create users and record what exercise they performed. This app uses MongoDB. So, CRUD functions are available.

####Used technologies:

  1.HTML
  2.CSS
  3.Bootstrap
  4.JavaScript
  5.React
  6.Express.js
  7.Node.js
  8.MongoDB
